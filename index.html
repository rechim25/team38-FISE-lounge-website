Abstract

FISE Lounge

COVID-19 has highlighted society's need to help people in social isolation, particularly amongst our most vulnerable â€“ the elderly. This project aims to help mitigate the potential health consequences for those in shielding by providing an interactive and privacy-safe video calling platform - proof of concept - made with the elderly in mind.

We have designed - by following guidelines and principles from gerontological studies - a privacy safe and customisable online simulated lounge as a multi-device front end ecosystem. It features open-source video calling and social interactivity applications, that make it easier for those in shielding to maintain contact with their dearest.

Next

Lounge App

For the elderly and those in shielding.

The Lounge acts as a smart, privacy-safe, and interactive video-calling app for the elderly and others in shielding. It provides a simple interface tailored for older people, and is intended to be used in the COVID-19 pandemic and other situations where social isolation is a major issue. Using this, the elderly can do everything from sending emergency messages and video-calling contacts to playing videogames and reading the latest news.

Next

Lounge Dashboard

An administrative dashboard for relatives or friends.

The dashboard is a standalone administrative page that provides a tool for the more tech-savvy relatives or friends to set up the Lounge app and preferences on the elderly's behalf. This includes adding contacts, customizing backgrounds and selecting privacy-settings. A One-Time-Code is then retrieved from the dashboard and given to the elderly for access to the main app. Dashboard completely removes the need for the older users or other consumers to have technical knowledge, as their account set-up is handled remotely.

Key Features

An interactive and privacy safe video-calling platform for those in shielding.

Privacy Safe

Federated on-device option available with complete locality of user data - no cloud processing unless selected.

Reliable Videocalling

Simple and reliable video-calling through Jitsi's open-source API.

No Complex Login

Consumers only need to enter a One-Time Code which then will be cached to the browser.

Emergency Messages

Send an emergency message to all your contacts when in need.

UI for the Elderly

Lounge's UI has been designed based on gerontological studies on web accessibility for the elderly.

Personalised Experience

As an admin, you can customise your dearest's experience with VR 360 (with effects) and static backgrounds photos.

Social Interactivity Apps

The lounge experience enables consumers to access the BBC live news feed, online games such as chess, embedded videos and website pages.

Extensible Plugins

Plugins have been designed to allow further development and enable clients to modify the application to suit their needs.

Voice Commands

Lounge has built-in support for voice commands through IBM Watson (cloud) and IBM FISE (federated).
